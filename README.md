# 哒哒提醒小程序

一个简洁美观的提醒类小程序，帮助你管理日常提醒事项。

## 功能特性

- ✅ **提醒列表**：清晰展示所有提醒事项
- ➕ **添加提醒**：快速添加新的提醒，支持设置提醒时间
- ✓ **完成提醒**：点击复选框标记提醒为已完成
- 🗑️ **删除提醒**：长按或点击删除按钮移除提醒
- 📊 **进度统计**：显示已完成/总提醒数量

## 使用说明

1. 在微信开发者工具中打开本项目
2. 点击"添加提醒"按钮，输入提醒内容
3. 可选择输入提醒时间（可选）
4. 点击复选框标记提醒为已完成
5. 点击"删除"按钮移除不需要的提醒

## 数据存储

提醒数据使用微信小程序的本地存储（`wx.setStorageSync`）保存，无需网络连接。
